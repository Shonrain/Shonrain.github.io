# 快速开始

前提条件

Node.jsopen in new window >= 16.15.0
Vueopen in new window >= 3.2.0
Ant Design Vueopen in new window >= 3.2.20

# 安装

  1. 设置 npm 镜像
npm set registry http://121.43.151.203/
  1. 安装
npm install -D miya-design

pnpm add -D miya-design

yarn add -D miya-design

# 使用

MiYa Design 支持按需引用（推荐）和全局引用两种方式，下面以使用热区组件的例子，分别对两种方式进行介绍

# 按需引用

<template>
  <div class="hot-zone-content">
    <hot-zone url="imageUrl" id="foo" v-model:zones="zones" />
  </div>
</template>

<script setup lang="ts">
import { HotZone } from 'miya-design';
import { type HotZoneItem } from 'miya-design/types';

const zones = ref<Array<HotZoneItem>>([]);
</script>


 









# 全局引用

  1. main.ts 中引入并使用
import miyaDesign from 'miya-design';

app.use(miyaDesign);


 
  1. vue 文件中直接使用
<template>
  <div class="hot-zone-content">
    <hot-zone url="imageUrl" id="foo" v-model:zones="zones" />
  </div>
</template>

<script setup lang="ts">
import { type HotZoneItem } from 'miya-design/types';

const zones = ref<Array<HotZoneItem>>([]);
</script>


 








类型

MiYa Design 中可以使用的类型，全都定义于 miya-design/types 中，按需引入即可
