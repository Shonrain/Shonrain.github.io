# UploadImage 图片上传

通过点击或者拖拽上传图片

本页中图片上传的例子都需要token，所以请先设置token

# 基础用法

设置好上传接口即可上传图片

上传图片
点击查看代码
<template>
  <upload-image
    :headers="{authorization: token || '', type: 'unioncenter'}"
    action="https://applets-dev.miyapay.com/api/ops/upload-oss-image"
    v-model="url"
  />
</template>

<script setup lang="ts">
  import { UploadImage } from 'miya-design';
  const url = ref('');
</script>
# 上传校验

可以对图片的尺寸、大小、格式进行校验

上传图片

图片尺寸要求为：500px * 500px

点击查看代码

    action="https://applets-dev.miyapay.com/api/ops/upload-oss-image"
    v-model="url"
    :valid-width="500"
    :valid-height="500"
  />
</template>

# 自定义样式
选择图片
点击查看代码
<template>
  <upload-image
    :headers="{authorization: token || '', type: 'unioncenter'}"
    action="https://applets-dev.miyapay.com/api/ops/upload-oss-image"
    v-model="url"
    :is-custom="true"
  >
    <template #customer>
      <div class="customer-upload-field" v-if="!url">
        <span class="choose-label">选择图片</span>
      </div>
    </template>
  </upload-image>
  <div class="img-content" v-if="url">
    <img :src="url" />
    <a @click="url = ''">删除</a>
  </div>
</template>

<script setup lang="ts">
  import { UploadImage } from 'miya-design';
  const url = ref('');
</script>

# Props

图片上传组件的属性说明如下：

属性说明类型默认值
action图片上传接口string"/api/ops/upload-oss-image"
value(v-model)图片链接string''
isCustom是否自定义样式booleanfalse
validWidth校验图片宽度number-
validHeight校验图片高度number-
maxM校验图片大小（MB）number-
maxKb校验图片大小（kb）number-
formats支持的图片格式Array<string>-
headers接口请求头中需要带的参数Object-
