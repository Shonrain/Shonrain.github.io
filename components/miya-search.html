# MiyaSearch 搜索栏

# 搜索栏支持如下组件：
  • 文本框
  • 选择器
  • 时间选择器
  • 数字范围
# 基本用法：
~
点击查看代码
<template>
  <div class="search-bar">
    <miya-search :items="items" :model="searchModel" />
  </div>
</template>

<script setup lang="ts">
import { MiyaSearch } from 'miya-design';
import { 
  FormSelectItem,
  newFormSelectItem, 
  newFormInput, 
  newFormSelect, 
  newFormDate, 
  newFormScale, 
} from 'miya-design/types';

const options: Array<FormSelectItem> = [
  newFormSelectItem('全部', -1),
  newFormSelectItem('启用', 1),
  newFormSelectItem('禁用', 0),
];

const items = [
  newFormInput('备注', 'remark', '请填写页面备注'),
  newFormSelect('状态', 'status', options),
  newFormDate('时间', 'createTime'),
  newFormScale('数字范围', 'scale', [0, 1], [100, 200], [1, 3]),
];

const searchModel = ref({
  remark: undefined,
  createTime: undefined,
  status: undefined,
  scale: [null, null],
})
</script>

# 控制每一行显示的组件数量：
~
点击查看代码
<template>
  <div class="search-bar">
    <miya-search :items="item5" :model="searchModel" :span="6" :gutter="8" />
  </div>
</template>

<script setup lang="ts">
import { MiyaSearch } from 'miya-design';
import { newFormInput, newFormSelect, newFormScale } from 'miya-design/types';

const items = [
  newFormInput('id', 'id', '请填写页面id'),
  newFormInput('备注', 'remark', '请填写页面备注'),
  newFormSelect('状态', 'status', options),
  newFormScale('范围', 'scale', [0, 1], [100, 200], [1, 3]),
];

const searchModel = ref({
  id: undefined,
  remark: undefined,
  status: undefined,
  scale: [null, null],
})
</script>

说明

span 值必须为 24 的约数，否则显示会出现错乱，即：

span = 8：一行三个

span = 6：一行四个

...

# 类型说明

# FormItem

FormItem 为搜索栏中组件的类型，搜索栏中的组件都是以该类型为基础构建出来的。

export interface FormItem {
  type: FORM_TYPE, // 组件类型，枚举
  label: string, // 标签
  prop: string, // 属性
  placeholder?: string | Array<string>, // 提示
  min?: [number | undefined, number | undefined], // 渲染数字范围时数字文本框中的最小值
  precision?: [number | undefined, number | undefined], // 渲染数字范围时数字文本框中的精度值
  options?: Array<FormSelectItem>, // 选择器组件中的子项
  mode?: 'multiple' | 'tags', // 选择器的模式，与antd中select组件中的mode一样
  format?: string, // 渲染日期组件时时间的格式
  showTime?: boolean, // 渲染日期组件时是否显示时间
  valueFormat?: string // 同antd中date-picker 中的 valueFormat
}

根据上面的例子可以看到，搜索栏中的组件，都是通过 props 中的 items 属性来进行渲染的。items 为一个数组，数组中的每个子项可以通过提供的一些方法来构建：

# newFormInput 文本框
const newFormInput = (label: string, prop: string, placeholder?: string) => FormItem

3 个参数：

  • label：标签
  • prop：属性（与 model 中的字段一致）
  • placeholder：文本框里显示提示信息（可选）
# newFormSelect 选择器
 const newFormSelect = (
  label: string, 
  prop: string, 
  options: Array<FormSelectItem>,
  placeholder?: string,
  mode?: 'multiple' | 'tags',
) => FormItem

5 个参数，其中 3 个参数与上面一样：

  • options： 下拉选框的子项：
export interface FormSelectItem {
  name: string,
  value: any,
}

// 单例方法
const newFormSelectItem = (name: string, value: any) => FormSelectItem
# newFormDate 日期选择
const newFormDate = (
  label: string,
  prop: string,
  showTime = false,
  format = 'YYYY-MM-DD',
  placeholder = ['开始时间', '结束时间'],
  valueFormat = undefined,
) => FormItem

6 个参数，其中 3 个参数与上面一样，placeholder 与上面类似，类型为[string, string]：

  • showTime：是否显示选择时间，这里的参数跟 antd 中 RangePickeropen in new window 组件的参数一样
  • format：日期格式
  • valueFormat：绑定值的格式，参考RangePickeropen in new window
# newFormScale 数字范围
const newFormScale = (
  label: string,
  prop: string,
  min?: [number | undefined, number | undefined],
  max?: [number | undefined, number | undefined],
  precision?: [number | undefined, number | undefined],
) => FormItem

4 个参数，其中 2 个参数与上面一样：

  • min：为一个长度为 2 的数组，数组的第一个参数表示前一个数字框的最小值，第二个参数为后一个数字框的最小值
  • max：为一个长度为 2 的数组，数组的第一个参数表示前一个数字框的最大值，第二个参数为后一个数字框的最大值
  • precision：为一个长度为 2 的数组，数组的第一个参数表示前一个数字框的精度，第二个参数为后一个数字框的精度
