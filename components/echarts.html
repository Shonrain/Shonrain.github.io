# ECharts 图表

适用于需要使用echarts的场景

# Components引入

# 目前Components的引入仅支持Bar、Line、Pie三种常见图表
# 基本用法：

点击查看代码
<template>
  <BarChart :series-data="data" :x-data="xData"  title="柱状图" />
</template>

<script setup lang="ts">
  import { BarChart } from 'miya-design';

  const xData = ref(['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'])
  const data = ref([120, 200, 150, 80, 70, 110, 130])
</script>
点击查看代码
<template>
  <LineChart :x-data="lineXData" :series-data="lineSeriesData" title="折线图" />
</template>

<script setup lang="ts">
  import { LineChart } from 'miya-design';

  const xData = ref(['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'])
  const data = ref([120, 200, 150, 80, 70, 110, 130])
</script>
点击查看代码
<template>
  <PieChart :series-data="data" title="折线图" />
</template>

<script setup lang="ts">
  import { LineChart } from 'miya-design';

  const data = ref([
    { value: 1048, name: 'Search Engine' },
    { value: 735, name: 'Direct' },

    { value: 484, name: 'Union Ads' },

  ])
</script>

# Props

属性说明类型默认值支持
xDatax坐标轴的值array[]Line、Bar
seriesDatay坐标轴的值array[]Line、Bar、Pie
title标题string''Line、Bar、Pie
width图表宽度string100%Line、Bar、Pie
height图表高度string300pxLine、Bar、Pie
theme主题stringdefault 【dark, light可选】Line、Bar、Pie
smooth平滑折线booleanfalseLine【仅支持单条】
multiple多条折线booleanfalseLine
