# 测试

当我们将打包好的文件发布到本地的 npm 仓库之后，就可以在测试工程中进行引用并测试了。

  1. 修改 package.json 中 miya-design 的版本号：
"dependencies": {
  "miya-design": "latest version",
}
  1. 在测试目录下执行：
pnpm install
  1. 最新版本安装成功之后，我们就可以在项目进行引入并进行测试了：
<script lang="ts" setup>
import { HotZone } from 'miya-design';
import { type HotZoneItem } from 'miya-design/types';

const zones: Ref<Array<HotZoneItem>> = ref([]);
</script>

<template>
  <HotZone url="imageUrl" v-model:zones="zones" id="hot-zone" />
</template>
