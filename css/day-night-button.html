实现

  • 解构视觉稿

通过观察视觉稿，我们可以发现整个 Button 主要由 4 部分构成：

  1. 圆形按钮 ☀️
  2. 圆形按钮周围的光晕
  3. 云朵 ☁️
  4. 夜晚模式下的星星 ⭐️

将视觉稿分解为上述几个模块之后，整个 Button 就没有那么复杂了。那么我们接下来就各个击破，分别去实现上述的几个模块。

  • 容器

  • 圆形按钮 ☀️

实现思路：

  1. 两个圆形按钮在 DOM 层面是独立的：day-button 和 night-button。通过外层的 circle-button-content 包裹
  2. 两个按钮彼此重合，在模式切换的时候设置不同的透明度，使得过渡不会那么突兀
  3. 按钮的位置通过移动外层 circle-button-content 实现
